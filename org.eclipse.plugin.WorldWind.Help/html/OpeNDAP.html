Remote OpenDAP Access

World Wind Geo Allows you to access remote data through the OpenDAP protocol. To open a remote data set simply:

  • Open the built-in web browser to a OpeNDAP server.
  • Click the link of the data set you wish to look at.
  • The data set will be automatically opened within the analysis view.
Note: Only NetCDF and GRIB files served through OpeNDAP can be opened!

Try out any of these remote catalogs:

  • UCAR THREDDS catalog: http://motherlode.ucar.edu:8080/thredds/catalog.html
  • NVODS (National Virtual Oceanographic Data System) http://ferret.pmel.noaa.gov/thredds/dodsC/las/catalog.html
